# 真人中文说话清单

## 使用时机

当用户指出文本仍有“AI 味”“机器腔”“不够像真人”“小红书不像真人博主”时，读取本文件。目标不是把文本改得粗糙，而是减少**过度规范、极度中立、结构模板化、情感克制、信息密度低**造成的平均感。

## 一眼识别：中文 AI 语风

中文 AI 语风常见问题是：书面化、模板化、零度情感、空洞正确、安全优先。自然人的中文更常出现口语化、情绪波动、个人印记、细节毛刺、非线性节奏和局部留白。

| 维度 | AI 常见表现 | 真人化修法 |
|---|---|---|
| 用词 | 标准书面词、大词、四字格、套话 | 换成日常词；能说“越来越”就别说“愈发”；能说“很多”就别说“诸多” |
| 句式 | 句长均匀，模板固定，翻译腔 | 长短错开；允许半截话、省略、轻微回头修正 |
| 结构 | 总分总、模块化、每段承担同样功能 | 允许跳一下；先讲最有感的点，再补信息 |
| 情感 | 永远温和、客观、全面 | 有偏向，有犹豫，有吐槽，但不编经历 |
| 信息 | 抽象判断多，细节少 | 多写具体对象、动作、数字、时间；删空道理 |
| 个性 | 千人一面，没有口头习惯 | 保留轻微个人措辞，但不要硬塞网络热词 |

## 禁用或高风险词

优先删除或替换这些词，除非原文语境确实需要：赋能、闭环、沉淀、深耕、多维、全域、态势、落地、助推、赋能升级、提质增效、协同共进、多措并举、多措并行、究其本质、换言之、从本质上来说、在当下语境下、放眼长远、兼具价值、极具意义、深度融合、生态、底层逻辑、价值沉淀、全面升级、深刻洞察、稳步推进。

连接词同样要谨慎：首先、其次、再者、与此同时、此外、综上、综上所述、由此可见、值得一提的是、不得不说、客观而言。社媒短文里能删就删，靠语义自然承接。

## AI 句式替换

| AI 写法 | 真人中文写法 |
|---|---|
| 在 XX 背景下，通过 XX 实现 XX | 现在 XX，直接 XX 就行 |
| 不仅……而且……更进一步 | 不光……还……甚至…… |
| 具备极高实用价值与深远意义 | 挺好用，也实在 |
| 从多个维度进行综合分析 | 换几个角度想想 |
| 整体呈现稳步向好发展趋势 | 整体慢慢变好了 |
| 存在一定优化提升空间 | 还有不少能改的地方 |
| 我最强的感觉是…… | 我感觉最强烈的是…… |
| 这意味着…… | 换句话压掉：直接写“以后可能会……”或“问题在……” |

## 口语颗粒度

口语化不等于把所有句子切碎。中文社媒和聊天可以出现短句、省略和轻微回头修正，但成熟内容号通常仍以中长句承载信息，只在判断处收短。允许“但说真的”“这点挺烦”“有点微妙”“先别急着下结论”这类轻微个人化表达。口语助词可以用，但要少量：啦、嘛、呗、其实、反倒、说白了。不要为了装人而连续使用“讲真、不得不说、你品、懂的都懂”。更不要把去 AI 味理解成降低文本质量、制造错别字或强行碎片化。

开头和结尾是最容易被写成“总结句”的位置。处理资讯短评、科技评论、社媒长文时，优先用更像真人说话的结构：先说“今天不用急着记住每个发布点”“我会先把它看成一个信号”“更值得盯的是”这类自然入口，再落到具体判断。不要用“我会记成一句话”“总结来看”“一句话概括”等显得刻意收束的表达。

## 情绪和个人印记

真人不必一直中立。可以明确写“我更在意”“我不太喜欢”“这条我会先观望”“这点有点夸张”。但不得伪造个人经历、朋友爆料、内部消息或现场见闻。没有真实素材时，用“我看下来”“我更关心”“我会先看”这类观察型表达，不写“我今天在现场”“我朋友说”。

## 节奏原则

不要每段都像“事实 + 解释 + 升华”。社媒短文可以先丢判断，再补事实；也可以先吐槽，再解释。成熟内容号的节奏通常是：先给一个真实入口，再用 2–4 句把信息和判断揉在同一段里，最后停在一个具体判断、一个提醒，或一个未解决的问题上。删掉空洞结尾，不要用“未来已来”“具有重要意义”“值得期待”。

## 成熟内容博主节奏

这套规则适用于小红书、公众号、长微博、科技资讯短评等场景，也可作为所有中文终稿的全局质检项。它强调的是**顺滑、有判断、有修辞感**，不是单纯口语化。

| 位置 | 推荐写法 | 应避免 |
|---|---|---|
| 开头 | 从真实感受、最反常的事实或一个具体判断进入，如“今天不用急着记住每个发布点”“我感觉最强烈的是……”“更值得盯的是……”；先抓读者注意力，再补事实 | “在当今时代”“随着技术不断发展”“我最强的感觉是”“一句话概括”等翻译腔或总结腔入口 |
| 段落 | 2–4 句为主，每段至少有一个事实、动作、对象或判断；允许一段中同时完成信息补充和态度表达 | 每段都只有一句口号，或每段都按“事实—意义—展望”机械闭合 |
| 情绪 | 克制但有偏向，可以写“我更在意”“这点有点微妙”“我会先观望”；用判断带情绪 | 全程中立、安全、正确，或用夸张惊叹代替分析 |
| 转场 | 用“先说说”“反观”“再聊”“这事有意思在于”“问题在这儿”“更关键的是”自然承接 | “首先、其次、此外、综上、值得注意的是”等讲义式连接 |
| 结尾 | 用口语化收束停在具体判断、现实限制、后续变量或一个没解完的问题上，如“更值得记住的是……”“真正要看的还是……”“这才是下一轮更现实的考题” | 升华到时代、格局、未来，或用“值得期待”“意义重大”“我会记成一句话”收口 |

如果文本涉及新闻、科技资讯、商业动态或未确认消息，真人感不能靠把语气写死。应保留“据报道”“可能”“尚未确认”“目前看”等限定词，既有判断，也不越过事实边界。

## 快速检查

改完后问三件事：第一，这句话一个中文母语者会不会顺嘴这么说；第二，这段有没有非必要的总分总和连接词；第三，作者有没有一点真实偏好，而不是永远正确、全面、安全。
