# 中文 AI 痕迹检测器

## 使用时机

当用户要求“去 AI 味”“真人化”“检查 AI 痕迹”“终稿打磨”或文本显得像模型回答时，读取本文件。目标是找出最影响读感的模式，并做**最小有效修补**，而不是机械替换全部词语。

## 总体判断

AI 味通常不是单个词造成的，而是**过度规范、极度中立、结构模板化、情感克制、信息密度低**共同形成的读感。检测时先判断文本的语域，再看哪些痕迹与该语域冲突。自然中文允许口语化、情绪波动、个人印记、细节毛刺和非线性节奏；不要把所有句子都修成安全、全面、正确、无争议。

| 维度 | 高风险表现 | 修补方向 |
|---|---|---|
| 信息 | 抽象判断多，具体事实少 | 增加原文已有的对象、时间、动作、因果；不要虚构事实 |
| 结构 | 三段式、总分总、每段同长度 | 合并弱段，打乱机械节奏，减少标题和列表 |
| 语气 | 永远平衡、积极、圆滑 | 恢复作者判断，允许有取舍、停顿和局部不确定 |
| 词汇 | 路标词、宏大词、宣传词密集 | 改成具体动作、具体关系和普通中文 |
| 互动 | “你会发现”“我们需要认识到”过多 | 去掉讲课姿态，直接陈述或换成场景化表达 |

## 高频 AI 痕迹

| 类型 | 典型信号 | 风险 | 改法 |
|---|---|---|---|
| 路标词堆叠 | 首先、其次、此外、值得注意的是、综上所述 | 像答案提纲，不像自然成文 | 删除或改成具体承接，如“问题出在”“更麻烦的是” |
| 二分对照壳 | 不是 A，而是 B；不仅是 A，更是 B | 过于像模型制造深度 | 直接说 B，必要时保留一个关键对照 |
| 万能拔高 | 这不仅改变了……也重塑了…… | 空泛，像结尾模板 | 回到具体影响：谁改变了什么，代价是什么 |
| 挑战机遇套话 | 挑战与机遇并存、不断演变的格局 | 商业白皮书腔 | 拆成具体风险和具体机会 |
| 讲义腔 | 我们需要理解、接下来我们将探讨 | 像课堂或客服 | 直接进入论点，不预告文章动作 |
| 伪客观 | 可以说、某种程度上、不可否认的是 | 怕担责，语气软 | 能确定就直说；不能确定就说明条件 |
| 过度平衡 | 一方面……另一方面…… | 缺乏真实判断 | 保留主次关系，不强行对称 |
| 三连排比 | 更高效、更智能、更便捷 | 广告腔和模型腔 | 只保留最有信息量的一项或改成句子 |
| 粗体和标题泛滥 | 每段都有小标题和强调 | 像 SEO 文档 | 长文可保留必要标题，散文/评论少用 |
| 情绪空话 | 令人深思、值得期待、具有重要意义 | 没有真实感受 | 写具体不适、惊讶、犹豫或判断依据 |
| 大词套话 | 赋能、闭环、沉淀、深耕、多维、全域、落地、生态、底层逻辑 | 正确但无实义，像企业文案 | 换成具体动作：谁做了什么，影响谁，代价是什么 |
| 中文句序别扭 | 我最强的感觉是、这个产品的起点是 | 像英文句法硬套中文 | 改成顺口表达：我感觉最强烈的是、这个产品从……开始 |
| 伪口语 | 说白了、讲真、不得不说 | 刻意装人 | 只在聊天/社媒语域少量使用 |
| 金句收尾 | 未来已来、答案就在其中 | 像生成式结尾 | 以具体动作、未解决问题或自然停顿收尾 |

## 英文句式渗透（LLM 系统性偏差）

LLM 的训练语料以英文为主，生成中文时底层句式逻辑仍是英文的。这不是翻译腔，而是更深层的句式逻辑问题。英文惯用抽象名词做主语、系动词做谓语；中文母语者惯用人或具体事物做主语、动词做谓语。这个偏差在所有 LLM 生成的中文里都存在，是 AI 味的根源之一，L1 改写时必须专门处理。

检测方法：扫描全文"X是Y"句式，判断是否在描述动作或状态（而非真正下定义）。如果是，把主语改成人或具体事物，谓语改成动词。判断标准是：这个句子，如果一个中文母语者在聊天里说出来，会不会觉得别扭？

| AI 生成（英文句式渗透） | 中文母语者的写法 |
|---|---|
| 这个产品的起点是一个情绪时刻 | 这个产品从我身边捕捉到的一个情绪中诞生 |
| 核心价值在于激发用户的主动感 | 用完之后用户会觉得"我能说" |
| 这个洞察的关键结论是…… | 这个洞察告诉我…… |
| 产品的定位转变是必要的 | 我把产品的定位改了 |
| 这一决策的背后逻辑是…… | 我这么做，是因为…… |
| 用户的核心诉求在于…… | 用户真正想要的是…… |

## 高风险词与连接词

社媒、评论和聊天语域中，以下词应默认删除或替换：赋能、闭环、沉淀、深耕、多维、全域、态势、落地、助推、提质增效、协同共进、多措并举、究其本质、从本质上来说、在当下语境下、放眼长远、兼具价值、极具意义、深度融合、生态、底层逻辑、价值沉淀。连接词中，首先、其次、再者、与此同时、此外、综上、由此可见、值得一提的是、客观而言也要谨慎，能靠上下文自然承接就不要加。

## 中文标点与节奏风险

冒号、分号、破折号和括号不是禁用项，但高频出现会显得像模型在整理概念。正式文本中冒号可用于定义、引用和清单；评论与随笔中要谨慎。连续短句会显得像“故意有力”，连续长句则容易像翻译腔。自然中文通常需要长短句交错，并且每段承担不同功能。

## 评分维度

| 维度 | 问题 | 低分信号 |
|---|---|---|
| 真实感 | 像一个有立场的人在写吗 | 全文安全中立，没有取舍 |
| 保真度 | 是否保留原文事实和边界 | 新增经历、数据、来源 |
| 语域一致 | 像目标场景里的中文吗 | 邮件像公众号，评论像报告 |
| 信息密度 | 是否有具体对象和动作 | 抽象名词连续堆叠 |
| 节奏 | 句长和段落是否自然 | 每段长度相近，三连结构过多 |

低于 35/50 时，应重写重点段落。35–42/50 时，做局部补丁。42/50 以上时，只做轻微修整，避免越改越模板。

## 审阅输出模板
